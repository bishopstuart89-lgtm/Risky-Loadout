CCJPA Web Schedule Validator

How to use this tool

What is this tool? This application allows the CCJPA team to upload raw GTFS transit data and instantly generate responsive, accessible HTML tables matching the Capitol Corridor brand. You can use this to visually validate schedules before sending data to the mobile app, or to generate the HTML needed for the public website.
Which files do I need to upload? You must select three specific files extracted from your GTFS Zip folder: stops.txt, trips.txt, and stop_times.txt.
How do I put this on the website? Once generated, click "Copy HTML Code" at the bottom of the page. You can paste that code directly into your CMS (like WordPress) to update the live schedule page.
