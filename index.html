CCJPA Web Schedule Validator

How to use this tool

What is this tool? This application allows the CCJPA team to upload raw GTFS transit data and instantly generate responsive, accessible HTML tables matching the Capitol Corridor brand.
NEW: Excel Comparison Feature You can optionally upload the Master Schedule Excel file (.xlsx or separated .csv sheets). The tool will cross-reference the GTFS timetable against the Excel sheet. Any times that don't match, or stops that exist in one but not the other, will be highlighted in red. Hover over a red cell to see the time the Excel document expected.
Which files do I need to upload? 1. The GTFS files: stops.txt, trips.txt, and stop_times.txt.
2. (Optional) The master Excel schedule file (e.g., Train Schedule_030226_ReturntoFullService.xlsx).
