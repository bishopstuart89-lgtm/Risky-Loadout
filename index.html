CCJPA Web Schedule Validator

How to use this tool

What is this tool? Upload raw GTFS transit data and the master Excel schedule. The tool generates a responsive, on-brand schedule table while cross-referencing both files to surface time mismatches, extra stops, and missing schedules. Hover any highlighted cell to read the expected value, or scroll down for the compiled Error Changelog.
Which files do I need to upload? 1. The GTFS files: stops.txt, trips.txt, and stop_times.txt.
2. (Optional) The master Excel schedule file with Weekdays Schedule and Weekend tabs.
How is the comparison performed? All parsing and diffing runs locally in your browser — files never leave your device. GTFS times within 60 seconds of the Excel value are considered matching (to accommodate minor dwell-time encoding differences between sources).
